# Healthcare Data Quality Analysis and Validation

## Objective
Perform data validation, cleaning, and reporting on a healthcare dataset.

## Steps Performed
- Checked missing values and duplicates
- Identified invalid data in multiple columns
- Cleaned and standardized data
- Visualized before vs after cleaning results

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Result
Reduced invalid data significantly and improved dataset quality for analysis.

## Project Files
- Jupyter Notebook: Healthcare_Data.ipynb
- Cleaned Dataset: train_cleaned.csv
